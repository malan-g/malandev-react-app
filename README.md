# Context
I built this client application to practice writing css transitions.

I built it using:
- React
- React Router 
- SASS modules & some Tailwind CSS

I've organized components like this:
```
component/
├── index.jsx
├── index.module.scss
├── animations.module.scss
├── data.js
└── components/
    ├── index.jsx
    ├── index.module.scss
    └── animations.module.scss
```
